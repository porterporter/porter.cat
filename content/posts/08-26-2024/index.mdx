Seen this error? You've changed the URL of your repository on git provider. Here's how to fix it.

```zsh
porter ❯ git remote get-url origin --all
https://github.com/porterporter/porter-cat.git
```

`git remote get-url origin --all` will list the origin url you're pushing to. When you change the URL of your repository, the old URL will warn you to switch it to the new one on git.

When you push, the new URL warning will actually tell you where to update it to as well

```zsh caption="This line right here"
remote: This repository moved. Please use the new location:
remote:   https://github.com/porterporter/porter.cat.git
```

To fix, just update your origin url with the command

```zsh
git remote set-url origin https://github.com/porterporter/porter.cat.git
```
